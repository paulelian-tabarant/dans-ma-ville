# Dans ma ville

## Démarrer l'API

Installer un environnement virtuel Python

```bash
python3 -m venv backend/venv
```

Activer l'environnement virtuel

```bash
source backend/venv/bin/activate
```

Installer les dépendances

```bash
cd backend
make install/dev
```

Lancer le serveur

```bash
make start
```

## Mettre à jour les dépendances

```bash
make update
```
