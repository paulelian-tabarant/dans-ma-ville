# Dans ma ville

## Démarrer l'API

```bash
cd backend
```

Installer un environnement virtuel Python

```bash
python3 -m venv venv
```

Activer l'environnement virtuel

```bash
source venv/bin/activate
```

Installer les dépendances

```bash
make install/dev
```

Lancer le serveur

```bash
make start
```

## Mettre à jour les dépendances

```bash
make update
```
